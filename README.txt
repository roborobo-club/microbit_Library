ロボロボClub(https://www.enjoy-robo.com/　)のmicro:bit用ライブラリを配布するためのデータセット

○Library_2023_Summer_Workshop_Set
 ・以下を含むライブラリセット
  ・Library_NXT_Motor
  ・Library_Photo_Reflector
  ・Library_PID 
  ・Library_TCS34725

 ・NXTのLモーターを使用するためのライブラリ
 ・ロボロボライブラリ「Library_PID」が必須
 ・拡張機能「pca9685」が必須

○Library_NXT_Motor_Set
 ・以下を含むライブラリセット
  ・Library_NXT_Motor
  ・Library_PID

○Library_Photo_Reflector
 ・フォトリフレクタを使用するためのライブラリ

○Library_PID
